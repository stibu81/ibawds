# Check All Links in the Slide Deck

Check links in all files of a slide deck using
[`check_links_in_file()`](https://stibu81.github.io/ibawds/reference/check_links_in_file.md).

## Usage

``` r
check_links_in_slides(path)
```

## Arguments

- path:

  path to the top level directory of a lecture

## Value

a tibble listing the links that did not work.
